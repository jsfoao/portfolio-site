## Overview

Nata Engine is a solo C++ project where I built gameplay and foundational engine systems with a strong focus on iteration speed and clean architecture.

## Core Work

- Collision and gameplay systems for fast prototyping
- Rendering-side architecture and debug tooling
- Data structures and update order design for predictable behavior

## Highlights

- Built with C++ and OpenGL
- Architecture designed for extending gameplay features quickly
- Focused on balancing engine depth with practical production workflows
