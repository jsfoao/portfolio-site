## Overview
Gameplay Framework that allows for quick prototyping and iteration of gameplay systems with special focus on multiplayer games.

## Highlights
* Object based input system separated into actions and contexts (collections of actions) that can easily be added and removed. Visualization tool to easily debug current contexts, action values, action modifiers, action triggers, etc.
* Multiplayer-ready modular context based interaction system where interactables can be easily extended by using interaction fragments. These are modular objects that define an interaction.
* Replicated gameplay ability system heavily inspired by Unreal Engine's GAS (Gameplay Ability System). Abilities are defined in lightweight, modular and decoupled objects.
* Systems are easy to use and setup. Quickly prototype and iterate mechanics.
